# Update Skill

Met à jour un skill existant en utilisant /skill-creator et context7, puis le partage dans un dossier communautaire avec commit + push.

```
J'aimerais que tu mettes à jour le skill /tanstack-query en utilisant le skill /skill-creator et context7 pour apprendre parfaitement tanstack-query, puis que tu le sauvegardes également dans le dossier /home/arthur/dev/skills en remplaçant l'ancienne version déjà présente pour le partager à ma communauté. Une fois fait, commit et push les changements en respectant conventional commit.
```
