# UX/UI Redesign PRD

Refonte complète de l'UX/UI d'un projet frontend. Explore les composants existants, s'inspire de designs via fontofweb, et produit un PRD structuré avec migration Radix vers BaseUI/CossUI.

## Generate PRD

Analyse les composants et génère un PRD complet de redesign UX/UI.

### FR

```
Explore les fichiers du dossier @src/components/, puis rédige un PRD dans le dossier @tasks en utilisant le skill /ralph-tui-prd pour refondre complètement l'UX/UI du codebase. Utilise les skills /web-design-guidelines et /frontend-design comme référence, et puisque tu migres de Radix vers BaseUI avec des composants CossUI, utilise le skill /coss-ui. Inspire-toi aussi des designs que tu trouveras dans le MCP fontofweb, et utilise context7 pour approfondir ta compréhension de Next.js, React et TailwindCSS.
```

### EN

```
Explore the files in the @src/components/ folder, then write a PRD in the @tasks folder using the /ralph-tui-prd skill to completely redesign the codebase UX/UI. Use the /web-design-guidelines and /frontend-design skills as reference, and since you're migrating from Radix to BaseUI with CossUI components, use the /coss-ui skill. Also draw inspiration from the designs you'll find in the MCP fontofweb, and use context7 to deepen your understanding of Next.js, React and TailwindCSS.
```
