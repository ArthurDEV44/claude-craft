# Design Exploration PRD

Explore une section UI existante, cherche de l'inspiration design via fontofweb, apprend la stack via context7, puis produit un PRD d'amélioration.

## FR

```
Explore le code des fichiers du dossier @src/components/landing. Utilise le serveur MCP fontofweb pour rechercher des designs inspirants, et context7 pour apprendre la stack technique utilisée dans ces fichiers. Enfin, rédige un PRD dans le dossier @tasks en utilisant le skill /ralph-tui-prd.
```

## EN

```
Explore the code in the files from the @src/components/landing folder. Use the MCP server fontofweb to search for inspiring designs, and context7 to learn the technical stack used in these files. Finally, write a PRD in the @tasks folder using the /ralph-tui-prd skill.
```
